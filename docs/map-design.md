# あやがわの夏 マップ設計書

実在の香川県綾川町の地理を下敷きにする。座標系: **+x = 北(下流・陶方面) / -x = 南(上流・綾上方面) / +z = 東 / -z = 西**。
綾川は綾上の山から北へ流れ、滝宮を抜けて陶の東をかすめる。ことでん琴平線が世界を斜めに横切る。

## 設計原則

1. **ゾーンごとに「役割」と「遊び」を必ず持たせる。** 見た目だけの場所を作らない。
2. **移動30秒ごとに何かがある。** 幹線を歩くと、視界に必ず次の目印(ランドマーク)が見える。
3. **分岐には道しるべ。** 迷子は探索の楽しさではなく苦痛。子どもの字の看板で誘導する。
4. **奥へ行くほど「秘密」が濃くなる。** 家の近く=日常、橋の向こう=冒険、山や竹やぶの奥=秘密。

## 道しるべ配置 (じゅんばんに増やす)

- 橋の北詰 (20, 0): 「←たきのみや / すえ→」
- 橋の南詰 (20, 26): 「↓てんまんぐう / ミナんち→」
- 商店通り東端 (-18, 24): 「←しょうてんがい / はし→」
- 陶の分かれ道 (126, 5): 「←すえのえき / ↑ほうじょういけ」
- 池の分かれ道 (129, 44): 「→とかめやま のぼりぐち」
- 綾上への道 (-95, 10): 「↑あやかみ (とおいよ)」

## 「秘密の濃さ」マップ

- レベル0 (いつでも): 河原、商店通り、ひまわり畑
- レベル1 (教えてもらう): クスノキのうろ (ケンタ友情5)、ウナギの主 (げんじい)
- レベル2 (この設計で追加): **竹やぶのひみつきち** (ケンタ友情4で招待)、**十瓶山の見晴らし台** (だれでも登れるが夕方だけ特別)
- レベル3 (最終週): 見晴らし台の夕日でケンタが本音を言う

## お盆の舞台 (8/13-15)

- 墓地: 陶の山すそ (142, 22) 付近。石垣と小さな墓石5基。ふだんは静かな場所として存在し、13日の朝にばあちゃんが誘ってくれる。
- 迎え火/送り火: おばあちゃんの家の門先 (120, 14)。13日夜と15日夜。

## 夕立 (ゆうだち)

晴れの日の一部で、14:15ごろから約40分だけ雨→あがると虹が北条池の方角にかかる。
「せみがピタッとやんで、あめのにおいがして、あがったら虹」の体験を1セットで。
